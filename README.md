## 最新完整版本（付费）
### [查看最新完整版本详细信息](https://github.com/SemperChen/tianxiashuge)
#### 最新完整版比开源版本更多功能：
- 支持微软的codepush热更新
- 支持极光推送
- 支持极光统计，统计安装用户、终端信息等
- 支持缓存书籍和左右滑动翻页功能
- 支持i18n，简体繁体阅读
- 可换皮肤，樱花粉、星辰黑一键切换
- 支持云书架，书籍阅读记录可保存到云书架
- 已更新到RN0.57.8，后期持续更新
